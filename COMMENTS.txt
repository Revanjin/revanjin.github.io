Ram Slots

	Steckplatzvorrichtung für Arbeitsspeicher
	Ram Austauschbar
	In der Regel 2 bis 4 Slots
		SDRAM, DDR 1-4
		DIMM = Anschlussart (Anzahl der Kontakte bei 168)
		
	Beispiel Anforderungen vom A770CrossFire:
		- Dual Channel DDR2 memory technology
		- 4 x DDR2 DIMM slots
		- Supports DDR2 1066*/800/667/533 non-ECC, un-buffered memory
		- Max. capacity of system memory: 16GB**

PCI / PCI-e
	
	Steckplatzvorrichtung für weitere Interne Geräte
	Peripheral Component Interconnect
		Bus-Standard für Verbindung
		Im Schnitt 2 bis 7 Slots
	PCI-Express
		PCI-Bus, aber nicht shared Bus-System
		(Netzwerkkarten, Modems, SCSI-Karten, Soundkarten, Grafikkarten usw.)
		https://upload.wikimedia.org/wikipedia/commons/thumb/9/9b/Prinzipielle_Darstellung_der_PCI-Architektur.svg/581px-Prinzipielle_Darstellung_der_PCI-Architektur.svg.png
		Mit Riser-Karte -> RAM Erweiterung
	Stromversorgung idr. via 8er Pins oder 6er Pins
